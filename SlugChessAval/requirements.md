# Visual Studio

# dotnetcore
You need dotnet 5.0 to build
To publish you need the dotnetcore tool DotnetVersion
Get it by running
>dotnet tool install -g DotnetVersion
NOTE: this tool requires 2.1 runtime.  >sudo apt install dotnet-runtime-2.1
